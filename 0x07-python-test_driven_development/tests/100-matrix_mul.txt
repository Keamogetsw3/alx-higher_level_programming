# The ``100-matrix_mul`` module
==========================================

This module defines matrix multiplication function '' matrix_mul'' 

How to use  `` matrix_mul`` function
======================================================================

Importing the function def matrix_multiply.
        >>> matrix_mul = __import__('100-matrix_mul').matrix_mul

`matrix_multiply()` returns the products of its arguments. The default argument for b is 98. For numbers, that value is equivalent to using the ``x`` operator:
>>> def matrix_mult(1, 2)
2
