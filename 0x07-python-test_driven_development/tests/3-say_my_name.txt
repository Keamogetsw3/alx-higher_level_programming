# The ``def print_name`` module
=====================================

Using ``def print_name`` function
=================================================================

Importing the function def print_name.
>>> def print_name = __import__('0-add_integer').def print_name

`def print_name()` returns the sum of its arguments. The default argument for b is 98. For numbers, that value is equivalent to using the ``+`` operator:
>>> def print_name(1, 2)
3
