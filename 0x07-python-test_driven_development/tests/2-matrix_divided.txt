# The ``def matrix_divided`` module
=========================================

Using ``def matrix_divided`` function
=====================================================================

Importing the function def matrix_divided.
>>> def matrix_divided = __import__('0-add_integer').def matrix_divided

`def matrix_divided()` returns the sum of its arguments. The default argument for b is 98. For numbers, that value is equivalent to using the ``+`` operator:
>>> def matrix_divided(1, 2)
3
