# The ``def text_indentation`` module
===========================================

Using ``def text_indentation`` function
=======================================================================

Importing the function def text_indentation.
>>> def text_indentation = __import__('0-add_integer').def text_indentation

`def text_indentation()` returns the sum of its arguments. The default argument for b is 98. For numbers, that value is equivalent to using the ``+`` operator:
>>> def text_indentation(1, 2)
3
